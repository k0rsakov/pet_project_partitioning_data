## О проекте

Запуск виртуального окружения для работы:

```bash
uv sync
```

Запуск `jupyter lab` через `uv`:

```bash
uv run jupyter lab
```

Для запуска контейнеров с PostgreSQL и ClickHouse, выполните команду:

```bash
docker-compose up -d
```
